**AI Resume Analyzer** is a web-based tool that provides instant, AI-powered feedback on your resume. It evaluates formatting, keyword usage, ATS compatibility, and offers suggestions to improve your chances of landing interviews.  

---

## Features
- Drag-and-drop or click-to-upload resume functionality  
- Supports PDF, DOC, DOCX, and TXT formats (up to 5MB)  
- Instant AI-generated resume score  
- Highlights strengths and areas for improvement  
- Suggests recommended skills and keywords  
- ATS (Applicant Tracking System) compatibility analysis  
- Responsive design with modern, animated UI  

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Design & Animations:** CSS keyframes, gradient backgrounds  
- **AI Integration:** Placeholder logic for AI analysis (replaceable with real API calls)  
- Fully client-side, no backend required for demo
